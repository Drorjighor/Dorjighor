# Final UI Polish QA

## Layout and Ratio

1. Hero slider maintains ratio on mobile, tablet, desktop.
2. Product cards use consistent image ratio.
3. Inner page spacing is consistent with page shell.

## Navigation and Routing

1. Navbar links navigate correctly on desktop and mobile.
2. Mobile drawer opens, closes, and overlay click works.
3. Direct route refresh does not show 404 on deployed site.

## Product Experience

1. Search from navbar lands on products with query applied.
2. Sort and filter controls work together.
3. New Products, Best Sellers, and Flash Deals sections render properly.

## Forms and Inputs

1. Login and signup forms validate correctly.
2. Checkout form handles required fields.
3. Return request and newsletter forms submit without UI glitches.

## Accessibility and Performance

1. Buttons and links have visible focus states.
2. Contrast is readable for badges and overlays.
3. First load feels responsive on average mobile network.

## Final Launch Gate

Ship only if all above checks pass in both:
1. Latest Chrome desktop.
2. Latest Chrome Android (or Safari iOS).
